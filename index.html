Password Manager

Login Screen

To log in there are 2 field: Username (or email) and password.
If you don't have an account you can make one, just by clicking the link below the login button.

Main Page

This is the main page. As you can see there are 4 main section:
title bar, application bar, display account panel and access/modify account panel.

Title Bar

Here you can find 2 menu.
With the first you can edit and manage your account and data. The Help bar gives you information about different part of the app.

Application Bar

In this section you can with many thing!
The letter represent the index of your account list (where you are right now).
If you click the first image you can add a new account to your library.
When you click on the locker it will be opening a new window with the password generator.
The last, and most important, thing that you can find in this bar is the search bar. You can type any website name that you have in your library and the app will show it to you.

New Account

These are the fields that you can compile while making a new account. By default the characters from the password and the ping field are not visible to the user, but you can trigger the eye botton to see them.

Password Generator

You can generate strong password with this feature. Choosing from lowercase letters, uppercase letters, numbers and simbols. You can select just some of them or all. The default password length is set to 12 but you can lower it to 1 or make it 'til 32 characters.
The password will be shown in the text field. You can copy it with the copy button, or if you don't like it you can make another one, just by clicking the generate button.

Account

If you click on the image or on the name of the website, it will automatically open into your browser.
There are 3 buttons that you will need to copy the password, modify the account credentials and to delete the account.
